About This Project

This team portfolio is an academic assignment for our Software Engineering module at ENSIA. The project aims to develop our skills in collaborative development, version control with Git and GitHub, and modern web technologies while showcasing our team's capabilities.

Learning & Growth

We embrace new technologies and methodologies to expand our software engineering knowledge.

Team Collaboration

We practice effective teamwork through Git workflows, code reviews, and shared responsibility.

Best Practices

We're committed to following industry standards and writing clean, maintainable code.

Version Control

We master Git and GitHub to manage our codebase efficiently and track project progress.
